1. Difference Between map(), filter(), forEach() & find() ?

ANSWER :

In JavaScript, map(), filter(), forEach(), and find() are all higher-order array methods that allow developers to manipulate arrays in different ways. Here are the differences between them:

map(): map() applies a callback function to each element of an array and returns a new array with the same number of elements, where each element is the result of the callback function. It does not modify the original array.

const numbers = [1, 2, 3, 4, 5];
const multipliedNumbers = numbers.map(number => number * 2);
console.log(multipliedNumbers); // Output: [2, 4, 6, 8, 10]

filter(): filter() applies a callback function to each element of an array and returns a new array with only the elements that pass a certain condition defined by the callback function. It does not modify the original array.

const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(number => number % 2 === 0);
console.log(evenNumbers); // Output: [2, 4]

forEach(): forEach() applies a callback function to each element of an array and does not return anything. It simply executes the callback function on each element of the array. It does not modify the original array.

const numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2)); // Output: 2 4 6 8 10

find(): find() applies a callback function to each element of an array and returns the first element that passes a certain condition defined by the callback function. If no element passes the condition, it returns undefined. It does not modify the original array.

const numbers = [1, 2, 3, 4, 5];
const evenNumber = numbers.find(number => number % 2 === 0);
console.log(evenNumber); // Output: 2
